# Instruction-Level-Simulator-for-x86-ISA-Subset
he Instruction-Level Simulator for x86 Subset (ISLx86) is a C++ program that takes the  inputs of an assembled text-file of x86 instructions or data, and fetches/executes each instruction sequentially. After each instructions execution, the state of the machine is dumped to file **run.dump** and the memory contents are dumped to file **mem.dump**.
